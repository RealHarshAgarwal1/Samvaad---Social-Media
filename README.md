# Samvaad 🤖

Samvaad is a premium Instagram clone built with the MERN stack (MongoDB, Express, React, Node.js). It features a sleek, "robotic" aesthetic and real-time social interactions.

## ✨ Features

- **Real-time Notifications**: Get instant alerts when someone likes your post or follows you.
- **Explore**: Discover a grid of posts from across the platform.
- **Search**: Functional user search by username or email.
- **Messaging**: Real-time chat with other users.
- **Profile Management**: Customize your bio and profile picture.
- **Social Interaction**: Like, comment, follow, and bookmark posts.

## 🚀 Tech Stack

- **Frontend**: React, Redux Toolkit, Tailwind CSS, Lucide React, Shadcn UI.
- **Backend**: Node.js, Express, Socket.io, Mongoose.
- **Storage**: Cloudinary for images.
- **Database**: MongoDB Atlas.

## 📦 Installation

1. Clone the repository.
2. Install root dependencies: `npm install`
3. Install frontend dependencies: `npm install --prefix frontend`
4. Create a `.env` file in the root directory with the following variables:
   - `MONGO_URI`
   - `SECRET_KEY`
   - `CLOUD_NAME`
   - `API_KEY`
   - `API_SECRET`
   - `URL` (Frontend URL)
5. Run the dev server: `npm run dev`

## 🌐 Deployment

This project is optimized for deployment on **Render** as a single web service. The backend automatically serves the built frontend files.
